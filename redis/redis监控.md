## 提供的原生监控

### 当前链接的客户端数和连接数

`redis-cli --stat`查看当前连接的客户端数，连接数等

```bash
------- data ------ --------------------- load -------------------- - child -

4          1.27M    6       0       17341 (+1)          111
4          1.27M    6       0       17342 (+1)          111
4          1.27M    6       0       17343 (+1)          111
```
